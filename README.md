# 🎓 UIU Friends Loan & Crowdfunding

A secure peer-to-peer **Loan Marketplace** and **Crowdfunding Platform** exclusively designed for students of **United International University (UIU)**.

The platform enables UIU students to request emergency loans, receive loan offers from fellow students, create crowdfunding campaigns, donate to verified campaigns, manage repayments, build trust through reviews, and communicate securely within the system.

---

## 📌 Project Overview

Many university students face temporary financial problems but hesitate to seek help from traditional financial institutions due to lengthy procedures or high interest rates.

UIU Friends Loan & Crowdfunding provides a trusted university-only ecosystem where verified students can:

- Request emergency loans
- Offer loans to fellow students
- Create fundraising campaigns
- Donate securely
- Build trust through ratings and reviews
- Manage repayments digitally

Only verified UIU students are allowed to participate in loan bidding, ensuring a trusted community.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/UIU-Friends-Loan-Crowdfunding.git
```

---

## Move Project

Copy the project folder into

```
xampp/htdocs/
```

Example

```
C:\xampp\htdocs\UIU Friends Loan & Crowdfunding
```

---

## Start XAMPP

Start

- Apache
- MySQL

---

## Import Database

Open

```
http://localhost/phpmyadmin
```

Create database

```
uiu_flc
```

Import SQL files from

```
db dump/
```

---

## Configure Database

Open

```
backend/config/db.php
```

Update credentials

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "uiu_flc";
```

---

## Run Project

Open

```
http://localhost/UIU Friends Loan & Crowdfunding/homepage.html
```
